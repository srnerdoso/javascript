O que é JavaScript?

  • JavaScript é uma linguagem de programação

    • Criada em 1995 por Brendan Eich para ser usada no Netscape
    • JavaScript e Java não são a mesma coisa
    • Seu nome oficial é ECMAScript
    • É uma linguagem de alto nível e interpretada
    • Multiparadigma

    • Principal linguagem de programação usada pelos navegadores web
    • Junto com HTML e CSS forma as três principais tecnologias da web
    • Pode ser usado fora dos navegadores através de runtimes como o Node.js
